Threat Hunt PCaps - 



Usage: 

* Put script in the same directory as the pcaps you want to analyze. 
* chmod +x the script. 
```
chmod +x ./threatHunt_pcaps.sh
```
* Run the script: 
```
./threatHunt_pcaps.sh 
```
